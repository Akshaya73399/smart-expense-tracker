# 💰 Smart Expense Tracker (Java MVC)

A **Smart Expense Tracker** built in **Java** using the **MVC (Model-View-Controller)** architecture. This console-based application helps users manage their daily expenses by recording transactions, categorizing spending, generating monthly reports, and storing data persistently using **Java Serialization**.

---

## 🚀 Features

- ➕ Add new expenses
- 📋 View all recorded expenses
- 📅 Generate monthly expense reports
- 💵 Calculate total spending
- 🗂️ Categorize expenses (Food, Travel, Shopping, Bills, Other)
- 💾 Persistent data storage using Java Serialization
- 🏗️ MVC Architecture for clean and maintainable code

---

## ⚙️ How to Run

1. Clone the repository

```bash

```

2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).

3. Compile and run:

```
ExpenseController.java
```
